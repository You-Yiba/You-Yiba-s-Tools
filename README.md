# 游一八工具箱

## 项目简介
游一八工具箱是一个集合各种实用小工具的网站，旨在让工作和生活更加便捷。



## 目录结构
```
You-Yiba's Tools/
├── assets/            # 静态资源文件
│   ├── css/           # 样式文件
│   └── js/            # JavaScript文件
├── tools/             # 工具目录
│   └── Task!/         # 任务管理工具
├── index.html         # 工具箱主页
└── README.md          # 项目说明文件
```

## 如何使用
1. 打开 `index.html` 文件即可访问工具箱主页
2. 在主页上点击对应工具卡片即可进入相应工具
3. 目前已实现的工具：
   - **Task!**：任务管理工具，帮助你追踪和完成各项任务

## 如何添加新工具
1. 在 `tools/` 目录下创建新的工具文件夹，例如 `NewTool/`
2. 在新文件夹中创建工具的主页面，例如 `NewTool.html`
3. 在 `index.html` 文件中添加新工具的卡片，参考现有工具卡片的结构
4. 确保新工具的设计风格与现有工具保持一致

## 技术栈
